🛒 Customer Shopping Behavior Analysis

📌 Project Overview
This project analyzes customer shopping behavior using a complete end-to-end data analytics workflow.
The goal is to extract actionable insights from customer purchase data and present them using interactive dashboards.

This project demonstrates skills in:

🐍 Python (Data Cleaning & EDA)
🗄 SQL (Data Querying & Transformation)
📊 Power BI (Data Visualization & Dashboarding)
📈 Business Insight Generation
